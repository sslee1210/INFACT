# INFACT Website

INFACT company website built with Vite, React, TypeScript, and Express.

## Requirements

- Node.js
- pnpm
- Git

## Local Development

```powershell
pnpm install
pnpm dev
```

## Build

```powershell
pnpm build
```

## GitHub Setup

After Git is installed and available in PowerShell:

```powershell
git init
git add .
git commit -m "Initial website source"
git branch -M main
git remote add origin https://github.com/YOUR_ACCOUNT/YOUR_REPOSITORY.git
git push -u origin main
```

Use a private GitHub repository if the site content or company reference materials should not be public.
